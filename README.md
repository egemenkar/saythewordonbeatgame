# Say The Word On Beat Challenge

**Say The Word On Beat Challenge** is a voice controlled rhythm game where timing and speech matter.  
Say the correct word exactly on the beat and keep the challenge going.

## About the App

The game uses real time voice recognition to detect your voice and timing.  
Players can customize the experience using themes and a custom game builder.
